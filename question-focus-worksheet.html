이름 
질문 초점(Question Focus)이란 학생이 스스로 질문을 만들어나갈 출발점을 교사가 설계·제공하는 것입니다. 좋은 질문 초점은 탐구의 나침반 · 메타인지 촉진 · 성취기준과 학습 목표의 연결이라는 세 가지 역할을 동시에 수행합니다. 아래 6가지 선정 기준을 익히고, 직접 질문 초점을 설계해보세요.

PART 4 질문 초점 사례 분석 — 무엇이 다른가?
❌ 개선이 필요한 질문 초점
"물은 어떤 상태로 변하나요?"
토론가능성 ✕ 개방성 ✕ 고차원적 사고 ✕
→ 단답형·사실 확인 수준. 학생이 탐구할 여지가 없고, 교과서 찾으면 끝남.
✅ 좋은 질문 초점
"물의 상태 변화는 우리 삶에서 어떤 문제를 해결하고, 또 어떤 문제를 만들어낼까?"
토론가능성 ✓ 전이가능성 ✓ 개방성 ✓ 고차원적 사고 ✓
→ 삶의 맥락·다양한 관점·후속 탐구 유도. 학생마다 다른 방향으로 질문 확장 가능.
❌ 개선이 필요한 질문 초점
"조선은 언제 건국되었나요?"
계속성 ✕ 확장성 ✕ 개방성 ✕
→ 정답이 하나. 탐구가 아닌 암기 질문. 후속 질문으로 이어지지 않음.
✅ 좋은 질문 초점
"조선이 500년 이상 지속될 수 있었던 가장 중요한 이유는 무엇이며, 그것이 오늘날에도 유효한가?"
토론가능성 ✓ 전이가능성 ✓ 확장성 ✓ 고차원적 사고 ✓
→ 현재와의 연결, 다양한 근거 경쟁, 후속 토론 자연스럽게 유도.
✍️ 위 사례를 참고하여, 내가 분석한 차이점을 정리해보세요.
PART 5 질문 초점 질(Quality) 평가 루브릭
평가 기준 우수 (3점) 보통 (2점) 노력 필요 (1점)
개방성·토론가능성 다양한 관점과 근거가 공존하며 학생 간 의견 충돌 가능 2~3가지 해석 가능하나 수렴 경향 있음 정답이 하나이거나 사실 확인 수준
전이·성취기준 연결 성취기준과 직결되며 타 교과·삶의 맥락으로 전이 명확 성취기준 연결은 되나 전이가 제한적 성취기준과 관련이 불분명함
계속성·확장성 단원 전체를 관통하며 3개 이상의 후속 질문 자연 생성 2~3차시 지속 가능, 후속 질문 1~2개 한 차시 안에서 종결됨
고차원적 사고 유발 분석·종합·평가 수준의 사고가 필수적으로 요구됨 적용 수준의 사고 유발, 고차적 사고는 부분적 기억·이해 수준 이하의 단순 사고
내 질문 초점 자기 채점:  개방성    /3  +  전이    /3  +  계속성    /3  +  고차사고    /3  =     /12
